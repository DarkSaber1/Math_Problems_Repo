## Task 1

# Solution

The sample space of the experiment is:

Ω = {ω₁, ω₂, ω₃, ω₄, ω₅}

Two events are defined as:

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

We determine the following operations on events.

### 1. Union of events (A ∪ B)

The union of two events contains all elementary outcomes that belong to **A or B (or both)**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

Combining all elements from both sets:

A ∪ B = {ω₁, ω₂, ω₃, ω₄, ω₅}

This union is equal to the entire sample space Ω.

---

### 2. Intersection of events (A ∩ B)

The intersection of two events contains only outcomes that belong **to both events simultaneously**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

The only common element is:

A ∩ B = {ω₃}

---

### 3. Difference of events (B \ A)

The difference B \ A contains elements that belong to **B but not to A**.

B = {ω₂, ω₃, ω₄}  
A = {ω₁, ω₃, ω₅}

Removing the common element ω₃:

B \ A = {ω₂, ω₄}

---

### 4. Difference of events (A \ B)

The difference A \ B contains elements that belong **to A but not to B**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

Removing the common element ω₃:

A \ B = {ω₁, ω₅}
